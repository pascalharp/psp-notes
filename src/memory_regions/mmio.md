# MMIO


| Start          | What        |
| -------------- | ----------- |
| `0x03000000`   | CCPV v5     |
| `0x03010104`   | Fuse        |
| `0x03010400`   | Timer 1     |
| `0x03010424`   | Timer 2     |
| `0x032000e8` \*| STS         |
| `0x03220000`   | SMN control |
| `0x03230000`   | x86 control |

_* `0x032000d8` for Zen 2_
